# Screenshot Examples for Bug Reports

This directory contains sample screenshots demonstrating various types of bugs and issues found during testing. These serve as examples for documenting visual defects.

## Screenshot Naming Convention

Screenshots should be named using the following format:
```
BUG-{NUMBER}-{description}-{timestamp}.png
```

Example: `BUG-001-password-spaces-20260206.png`

## Best Practices

1. **Capture Context** - Include full browser window
2. **Highlight Issues** - Use arrows/circles
3. **Include Details** - Browser console, network tab
4. **Before/After** - Show comparisons
5. **Multiple Angles** - Different browsers/devices
